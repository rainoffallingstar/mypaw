## рҹҡҖ еҝ«йҖҹжҺҘе…Ҙ

е°ҶдҪ зҡ„ API Base URL жӣҝжҚўдёәжң¬ Space ең°еқҖеҚіеҸҜпјҡ

```bash
# еҺҹе§ӢиҜ·жұӮ
curl https://api.anthropic.com/v1/messages ...

# жӣҝжҚўдёәжң¬д»ЈзҗҶ
curl https://[user_name]-[space_name].hf.space/v1/messages \
  -H "x-api-key: дҪ зҡ„и®ҝй—®еҜҶй’Ҙ" \
  -H "Content-Type: application/json" \
  -d '{"model": "claude-4-6-sonnet", "max_tokens": 1024, "messages": [{"role": "user", "content": "Hello"}]}'
```

### Python зӨәдҫӢ

```python
import anthropic

client = anthropic.Anthropic(
    api_key="дҪ зҡ„и®ҝй—®еҜҶй’Ҙ",
    base_url="https://[user_name]-[space_name].hf.space",
)

message = client.messages.create(
    model="claude-4-6-sonnet",
    max_tokens=1024,
    messages=[{"role": "user", "content": "дҪ еҘҪпјҒ"}]
)
print(message.content)
```

### OpenAI SDK е…је®№

```python
from openai import OpenAI

client = OpenAI(
    api_key="дҪ зҡ„и®ҝй—®еҜҶй’Ҙ",
    base_url="https://[user_name]-[space_name].hf.space/openai",
)

response = client.chat.completions.create(
    model="gpt-5.1",
    messages=[{"role": "user", "content": "дҪ еҘҪпјҒ"}]
)
```

## вҡҷпёҸ йғЁзҪІй…ҚзҪ®

### еҝ…йңҖзҺҜеўғеҸҳйҮҸ

еңЁ HuggingFace Space **Settings вҶ’ Secrets** дёӯй…ҚзҪ®пјҡ

| еҸҳйҮҸеҗҚ | иҜҙжҳҺ | зӨәдҫӢ |
|--------|------|------|

| `MANAGEMENT_PASSWORD` | з®ЎзҗҶеҗҺеҸ°еҜҶз Ғ | `your-secure-password` |

## рҹ”‘ ж·»еҠ  API Key

жңҚеҠЎеҗҜеҠЁеҗҺпјҢйҖҡиҝҮз®ЎзҗҶжҺҘеҸЈж·»еҠ дҪ зҡ„ API Keyпјҡ

```bash
# ж·»еҠ  Claude Key
curl -X POST https://[user_name]-[space_name].hf.space/management/clients \
  -H "Authorization: Bearer $MANAGEMENT_PASSWORD" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "my-client",
    "claude-keys": ["sk-ant-api03-xxxxx"]
  }'

# ж·»еҠ  Gemini Key  
curl -X POST https://[user_name]-[space_name].hf.space/management/clients \
  -H "Authorization: Bearer $MANAGEMENT_PASSWORD" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "my-client",
    "gemini-keys": ["AIzaSy-xxxxx"]
  }'
```
